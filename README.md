## Installation

- Using `vim-plug`:

```vim
Plug 'ntk148v/habamax.nvim'
```

- Using `packer`:

```lua
use { "ntk148v/habamax.nvim" }
```

- Using `lazy.nvim`:

```lua
{ "ntk148v/habamax.nvim" }
```
