## Installation

- Using `vim-plug`:

```vim
Plug 'rktjmp/lush.nvim'
Plug 'ntk148v/habamax.nvim'
```

- Using `packer`:

```lua
use { "ntk148v/habamax.nvim", requires={ "rktjmp/lush.nvim" } }
```

- Using `lazy.nvim`:

```lua
{ "ntk148v/habamax.nvim", dependencies={ "rktjmp/lush.nvim" } }
```
